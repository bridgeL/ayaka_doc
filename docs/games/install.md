## 安装

**通过nb-cli安装（推荐）**

`nb plugin install nonebot-plugin-ayaka-games`

**通过pip安装**

`pip install nonebot-plugin-ayaka-games`

## 配置

`ayaka_setting.json`首次运行bot后生成

修改`__root__/use_playwright`为false

```json
{
    "__root__": {
        "use_playwright": false,
        ...
    },
    ...
}
```

配置修改后需重启bot
