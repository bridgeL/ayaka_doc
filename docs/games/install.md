## 安装

`nb plugin install nonebot-plugin-ayaka-games`

如果缺失nb则请先安装nb-cli `pip install nb-cli`

## 配置

首次运行bot后生成 `ayaka_setting.json`文件，修改其`__root__/use_playwright`为false

```json
{
    "__root__": {
        "use_playwright": false,
        ...
    },
    ...
}
```

配置修改后需重启bot
